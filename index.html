CryptoLock

CryptoLock is a browser-based tool for secure file encryption and decryption. Users can encrypt files by uploading them and entering a password, which must be confirmed. The tool uses advanced encryption methods like AES-CBC and PBKDF2, ensuring that all operations are performed locally in the browser for maximum privacy. For decryption, users input the same password used during encryption. The application is designed to handle file encryption and decryption securely and intuitively, without any external network connections.

SimplyRead

This project is a web app that summarizes articles with one click using OpenAI. Built with React.js, TypeScript, Redux Toolkit, and Tailwind CSS, it offers a modern and user-friendly interface. Users can input an article URL to get a concise summary generated by AI. Summaries are saved locally for easy access, and the app features copy-to-clipboard functionality for sharing. It efficiently manages data fetching with Redux Toolkit Query, optimizing performance and user experience.
